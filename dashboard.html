Soul Characters

via show_characters · trained identities reusable across Soul models

No trained characters yet

A Soul Character is a face-faithful identity model trained on 5–20 reference photos. Once trained, generate with that exact likeness via generate_image using model: soul_2 and the returned soul_id.

  • Training takes up to ~10 minutes (runs in background).
  • Requires a paid plan (you're on basic — should be fine).
  • One-shot generations don't need training — use soul_2 or nano_banana_2 with a single reference instead.
